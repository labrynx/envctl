## When to use reference

Use Reference when your question sounds like:

- what is the exact command syntax
- what does this command guarantee or refuse to do
- how are profiles selected exactly
- what shape does user config have

If you are still learning the concepts, start in [Concepts](../concepts/index.md). If you want workflow guidance, use [Guides](../guides/index.md).
